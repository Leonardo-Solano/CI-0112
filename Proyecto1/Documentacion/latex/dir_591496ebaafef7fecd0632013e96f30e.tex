\hypertarget{README.md_autotoc_md1}{}\doxysubsection{\texorpdfstring{Descripción}{Descripción}}\label{README.md_autotoc_md1}
Este proyecto consiste en una aplicación de consola desarrollada en Java que permite al usuario elegir entre jugar dos juegos. La primera opción es el \doxylink{class_ahorcado}{Ahorcado} y e segundo Batalla Naval. Esto a través de un menú principal que permite al usuario elegir el juego que desea ejecutar.\hypertarget{README.md_autotoc_md2}{}\doxysubsection{\texorpdfstring{Estructura del Proyecto}{Estructura del Proyecto}}\label{README.md_autotoc_md2}
