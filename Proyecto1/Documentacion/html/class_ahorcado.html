◆ jugar()

void Ahorcado.jugar ( )
inline

Método que ejecuta una partida completa del juego.

Este método coordina la entrada de letras por el jugador, llama a los métodos de verificación y determina si se gana o se pierde. Solo termina cuando se adivina la palabra o se acaban los intentos.
